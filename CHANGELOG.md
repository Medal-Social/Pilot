# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.4] - 2025-04-07

### Fixed

- Use Node 24 and OIDC-only auth for trusted publishing
- Use OIDC trusted publisher instead of NPM_TOKEN

## [0.1.3] - 2025-04-06

### Fixed

- Use PilotError codes and hide internal details from Update screen

## [0.1.0] - 2025-04-01

### Added

- Initial public release
- AI-powered CLI with crew routing (Brand, Marketing, Tech, CS, Sales)
- Plugin system with manifest validation (@medalsocial/kit, @medalsocial/sanity, @medalsocial/pencil)
- Split-panel UI with TabBar navigation
- Design token system with light/dark theme support
- Smart update system with manifest.json checksums
